# Customer360_using_synapse

Build a single, trusted 360° view of customers to drive personalized marketing and data-driven decisions.

A structured end-to-end data platform to analyze customer behavior across multiple retail channels.

Tools: Azure Synapse Analytics, ADLS, Azure Data Factory, Delta Lake, Serverless SQL Pool, Power BI, Draw.io

Key Features:

Data Ingestion (Bronze Layer): Integrated raw customer data from online, in-store, service, and loyalty channels into ADLS, preserving source formats.

Data Cleaning & Validation (Silver Layer): Standardized schemas, applied quality checks, and curated datasets using Delta format for reliable analytics.

Slowly Changing Dimensions (Gold Layer): Implemented SCD Type-2 logic to track historical and current customer records.

Analytical Modeling: Built Delta tables and analytical views for KPIs like Average Order Value, segmentation, and purchase trends.

Visualization: Created interactive Power BI dashboards for self-service analytics and actionable insights.

Architecture: Designed end-to-end Bronze–Silver–Gold pipelines with Draw.io documentation for scalable, modular solutions.

Impact: Delivered a trusted 360° view of customers, enabling personalized marketing, enhanced loyalty strategies, and data-driven decisions.
