# Customer 360 Retail Analytics


**Build a single, trusted 360° view of customers to drive personalized marketing and data-driven decisions.**  

A structured end-to-end data platform to analyze customer behavior across multiple retail channels.

---

## Tools & Technologies
- **Data Platform:** Azure Synapse Analytics, Azure Data Lake Storage (ADLS), Delta Lake  
- **ETL & Pipelines:** Azure Data Factory (ADF) Data Flows, Serverless SQL Pools  
- **Visualization & Analytics:** Power BI  
- **Architecture & Documentation:** Draw.io  

---

## Key Features

- **Data Ingestion (Bronze Layer):**  
  Integrated raw customer data from online, in-store, service, and loyalty channels into ADLS, preserving source formats for traceability.

- **Data Cleaning & Validation (Silver Layer):**  
  Standardized schemas, applied quality checks, and curated datasets using Delta format to support reliable analytics.

- **Slowly Changing Dimensions (Gold Layer):**  
  Implemented SCD Type-2 logic to track historical and current customer records, maintaining full customer history.

- **Analytical Modeling:**  
  Built Delta tables and analytical views to monitor KPIs like Average Order Value (AOV), customer segmentation, purchase frequency, and peak trends.

- **Pipeline Architecture:**  
  Designed end-to-end Bronze–Silver–Gold pipelines with Draw.io documentation, ensuring scalability, modularity, and maintainability.

---

## Business Impact
Delivered a trusted 360° view of customers, enabling personalized marketing, improved loyalty strategies, and data-driven operational decision-making.
